## Navigator

包含浏览器的信息，通常用于检测浏览器与操作系统的版本。

### 属性

| 属性名         | 解释                      |
| ----------- | ----------------------- |
| appCodeName | 浏览器代码名字符串表示             |
| appName     | 返回浏览器名字                 |
| appVersion  | 浏览器平台和版本信息              |
| platform    | 运行浏览器的操作系统平台            |
| userAgent   | 返回客户机发给服务器user-agent头的值 |
